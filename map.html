🚍 Prim & Dijkstra Route Planner

Find shortest distance using Dijkstra and cheapest transportation network using Prim.

About This Project

This web application uses Dijkstra's algorithm to find the shortest distance between two places and Prim's algorithm to find the cheapest transportation network connecting all places.

Considered locations: Jayanagar, JP Nagar, NICE Road, and Jain University. There are two main paths from Jayanagar to Jain University:

  • Jayanagar → NICE Road → Jain University
  • Jayanagar → JP Nagar → Jain University
